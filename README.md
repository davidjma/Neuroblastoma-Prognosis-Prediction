## **Neuroblastoma Prognosis Prediction**

This respiratory is a collection of scripts that aided me in predicting neuroblastoma prognosis using MSK-IMPACT DICOM and niftii format images of CT and PET scans. Data is kept confidential. For project description and outcome: [Neuroblastoma Prognosis Prediction Overview PDF](https://raw.githubusercontent.com/davidjma/Neuroblastoma-Prognosis-Prediction/main/PET-CT%20model%20for%20Neuroblastoma%20progression.pdf).
