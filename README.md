## **Neuroblastoma Prognosis Prediction**

This respiratory is a collection of scripts that aided me in predicting neuroblastoma prognosis using MSK-IMPACT DICOM and NIFTII formated images of CT and PET scans. Data is kept confidential. Different models (ResNet Suspicion Model, Bone-mask Model, PET signal Mask Model) were used where highest AUROC of 0.7 was acheived. 

For project description and outcome: [Neuroblastoma Prognosis Prediction Overview PDF](https://raw.githubusercontent.com/davidjma/Neuroblastoma-Prognosis-Prediction/main/PET-CT%20model%20for%20Neuroblastoma%20progression.pdf).
